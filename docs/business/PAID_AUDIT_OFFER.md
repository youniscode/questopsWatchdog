# QuestOps Watchdog — Paid Audit Offer

This document describes the commercial audit service for game server operators who use QuestOps Watchdog.

---

## Service overview

A manual, expert review of your QuestOps Watchdog scan outputs, server configuration, and operational setup. Delivered as a PDF report with actionable recommendations.

## What the client receives

- **PDF audit report** with PASS/FAIL summary per server, trend observations, and risk assessment
- **Configuration review** — threshold tuning, missed checks, false-positive reduction
- **Security check** — webhook hygiene, file permissions, env var usage
- **Recommendations** — prioritised by impact (critical, high, medium, low)
- **30-minute follow-up call** (optional, remote) to walk through findings

## What we need from the client

1. The **audit results bundle** (generated by `export_questops_audit_results.ps1`)
2. A brief description of their server setup (games, player count, hosting type)
3. Any specific concerns or issues they have noticed

Turnaround: 2–5 business days depending on tier.

## Scope boundaries

- **Included:** Review of scan outputs, config analysis, security posture, written recommendations.
- **Not included:** On-site support, server configuration changes, Discord alert setup (covered by the free tool), ongoing monitoring.
- **Not included:** Analysis of game-specific configuration files, mod conflicts, or performance tuning.

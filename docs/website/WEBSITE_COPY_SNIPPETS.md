*No installation needed — QuestOps Watchdog is a set of `.ps1` scripts that run in PowerShell 5.1, which ships with every modern Windows version.*

*Unlike most monitoring tools, there's nothing to install. No agent, no service, no background process. Just scripts you run when you need them (or schedule via Task Scheduler).*

*Zero setup beyond downloading the zip and editing a JSON config. If you can open a text file and run a PowerShell command, you're ready.*

*No installation required on the client machine. The tool package is a zip — extract and run. No admin rights needed for basic scans.*
